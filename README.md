## 🧯 Troubleshooting

### The file will not open
- Make sure you downloaded all files
- Unzip the folder first if the file came in a ZIP
- Check that you opened the right file for Windows

### Windows blocks the file
- Right-click the file and check the properties
- If there is an **Unblock** box, select it
- Try running the file again

### Nothing seems to happen
- Wait a short time for the script to finish
- Open Task Manager and check for a running process
- Run the file again as administrator if needed

### The window closes fast
- Open Command Prompt first
- Run the script from inside the folder
- That can keep the window open so you can read the output

## 🔐 Safety and access

Some ops tasks need higher access on Windows. That helps the tool read system data or make repairs.

- Use admin access only when needed
- Close other work before repair steps
- Review each prompt before you allow it
- Save any open files before you run scans or fixes
